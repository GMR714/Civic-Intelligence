# Phase 01.1: Frontend and Mock Dashboards

## Context
This plan focuses entirely on the visual presentation layer to build a high-fidelity 'Pitch Deck' style prototype that is navigable. The goal is to mock out "Ipê City" Floripa-specific data, dashboard bento boxes, and the impact matrix, using static `CSS` and `framer-motion` to keep it extremely fast and premium-looking without heavy interactive data bindings yet.
